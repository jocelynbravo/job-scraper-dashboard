# Job Market Scraper Dashboard

A web scraping + data visualization project that collects job listings (Data Analyst, Data Scientist, BI Developer, Data Engineer) and displays insights through an interactive dashboard — built with Python and Chart.js.

---

## Live Demo

👉 [View Dashboard](https://jocelynbravo.github.io/job-scraper-dashboard)

---

## What This Project Does

1. **Scrapes** job listings using Python (`requests` + `BeautifulSoup`)
2. **Cleans** the data with `pandas` (removes duplicates, classifies work type & experience level)
3. **Saves** results to CSV and JSON
4. **Visualizes** insights in an interactive dark-themed dashboard

---

## Dashboard Features

- **4 KPI cards** — Total listings, average salary, % remote jobs, top skill
- **Listings by Role** — Bar chart comparing job volume per role
- **Avg Salary by Role** — Horizontal bar chart showing compensation ranges
- **Work Type Distribution** — Donut chart (Remote / Hybrid / On-site)
- **Top Skills** — Horizontal bar chart of most requested skills (SQL, Python, Power BI...)
- **Interactive table** — Search, filter by role/type/level/salary, sort by salary
- **5 live filters** — Role, Work Type, Experience, Min Salary, Keyword search

## How to Run the Scraper

```bash
# 1. Clone the repo
git clone https://github.com/jocelynbravo/job-scraper-dashboard.git
cd job-scraper-dashboard

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the scraper
python job_scraper.py

# 4. Open the dashboard
open index.html
```

The scraper will save results to `data/jobs_raw.csv` and `data/summary.json`.
